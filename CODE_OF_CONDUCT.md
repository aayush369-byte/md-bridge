# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc-2-1] as its
code of conduct.

The full text lives at the Contributor Covenant site:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

The short version: be respectful, assume good faith, give and receive
feedback without personal attacks. Project maintainers commit to acting on
reports fairly and confidentially.

## Reporting

If you witness or experience behavior that violates the code, contact the
maintainer at **vinicq@gmail.com**. Reports stay confidential.

## Scope

The code applies in every project space (issues, pull requests, discussions,
chats) and to public representation of the project elsewhere on the web.

[cc-2-1]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
